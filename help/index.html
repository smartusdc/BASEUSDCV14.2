Getting Started with BASEUSDC.COM

Prerequisites:
  • MetaMask wallet installed
  • Small amount of ETH (approximately $1-2) for transaction fees
  • Minimum 0.01 USDC to start staking

Complete Setup Guide

Step 1: Create Coinbase Account
  1. Visit Coinbase.com and complete registration
  2. Verify your identity (KYC process)
  3. Add a payment method (bank account/card)
  4. Purchase USDC and ETH
Step 2: Configure MetaMask
  1. Install MetaMask browser extension
  2. Create or import a wallet
  3. Add Base Network (details provided in Wallet Setup)
  4. Transfer USDC and ETH from Coinbase
Step 3: Begin Staking
  1. Connect wallet to BASEUSDC.COM
  2. Approve USDC contract interaction
  3. Enter staking amount
  4. Confirm transaction in MetaMask

Wallet Setup Guide

Adding Base Network to MetaMask
Network Details:
  • Network Name: Base
  • RPC URL: https://mainnet.base.org
  • Chain ID: 8453
  • Currency Symbol: ETH
  • Block Explorer: https://basescan.org
Transferring Funds to Base Network
  1. Copy your MetaMask wallet address
  2. Send USDC from Coinbase (minimum 0.01 USDC)
  3. Send ETH for gas fees ($1-2 worth)
  4. Wait for confirmations (5-10 minutes)

Rewards & APR Information

Current APR: 74%

APR rates are reviewed monthly based on market conditions and platform performance. Any changes are gradual and announced in advance.

Understanding Your Rewards

Reward Components:
  • Base APR: 74% on staked USDC
  • Referral Bonus: +70% APR when using a referral code
  • Referrer Rewards: 5% of referred users' rewards

Rewards are calculated continuously and can be claimed at any time. The platform uses a compound interest model where unclaimed rewards continue generating returns.

Reward Calculation Example

For 1,000 USDC staked:

  • Daily Reward (Base): ~$2.02 (74% / 365)
  • With Referral: ~$3.45 (144% / 365)
  • Monthly Estimate: ~$60.60 - $103.50

Note: Actual rewards may vary based on the duration of stake and any APR adjustments.

Referral Program Guide

Creating Your Referral Code
As a Referrer:
  1. Connect your wallet
  2. Click "Generate Referral Code"
  3. Share your unique code
  4. Earn 5% of referred users' rewards
Using a Referral Code
As a Referred User:
  1. Enter the referral code before staking
  2. Receive +70% APR bonus
  3. Bonus applies to all future stakes
  4. No minimum stake requirement
Important Notes:
  • Each wallet can only use one referral code
  • Referral relationships are permanent
  • Referral rewards are claimable separately from staking rewards
  • No limit on the number of users you can refer

Troubleshooting Guide

Before contacting support, please review these common issues and their solutions. Most problems can be resolved using this guide.

Transaction Issues

Transaction Pending for Long Time

Solution:

When a transaction remains pending for an extended period, follow these steps:

  1. Open MetaMask and check for pending transaction notifications.
  2. If the transaction is stuck, you can:
    • Speed up the transaction by increasing gas price
    • Cancel the transaction if it hasn't been mined
    • Reset MetaMask activity if necessary

Note: On Base Network, transactions typically confirm within 15 seconds. If a transaction is pending longer than 2 minutes, action may be needed.

Insufficient Gas Fee Errors

Error: [ethjs-query] while formatting outputs from RPC '{"value":{"code":-32603,"data":{"code":-32000,"message":"insufficient funds for gas"}}}'
Solution:

This error occurs when you don't have enough ETH for transaction fees. To resolve:

  1. Ensure you have at least $1-2 worth of ETH in your wallet for gas fees.
  2. Transfer ETH from Coinbase to your MetaMask wallet on Base Network.
  3. Wait for lower gas prices if they are unusually high (normal gas fees on Base are under $0.10).

Wallet Connection Issues

MetaMask Not Connecting

Common Causes and Solutions:
  1. Wrong Network:
    • Ensure you're on Base Network
    • Click "Switch Network" when prompted
    • Add Base Network manually if needed using the network settings provided above
  2. Browser Issues:
    • Clear browser cache and cookies
    • Disable and re-enable MetaMask extension
    • Try using MetaMask's built-in browser on mobile

USDC Approval Issues

USDC Approval Not Working

Steps to Resolve:
  1. Verify USDC Balance:
    • Check if you have sufficient USDC in your wallet
    • Ensure you're checking the Base Network balance, not other networks
  2. Reset USDC Approval:
    • Try approving a smaller amount first
    • If previous approval is stuck, wait 10 minutes and try again
    • Check if you have enough ETH for the approval transaction

Reward Calculation and Claiming

Rewards Not Showing or Incorrect

Understanding Reward Calculations:

Rewards are calculated based on:

  • Your staked amount
  • Current APR rate
  • Time since last claim or deposit
  • Any applicable referral bonuses

If rewards appear incorrect:

  1. Refresh the page and reconnect your wallet
  2. Verify your staking duration
  3. Check if any recent deposits are still pending
  4. Confirm your referral status if applicable

Emergency Solutions

Unable to Withdraw Funds

If you're experiencing withdrawal issues:
  1. Verify Contract Status:
    • Check the contract on BaseScan
    • Ensure withdrawals are not temporarily paused
  2. Technical Requirements:
    • Maintain sufficient ETH for gas fees
    • Try withdrawing a smaller amount first
    • Ensure you're using the correct wallet address
