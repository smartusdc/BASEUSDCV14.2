Wallet Connection Issues

MetaMask Not Connecting

Common Causes and Solutions:
  1. Network Configuration Issues:
    • Verify you are on Base Network in MetaMask
    • Select "Switch Network" when prompted
    • If needed, manually add Base Network using the configuration details provided in the Wallet Setup section
  2. Browser-Related Issues:
    • Clear your browser's cache and cookies
    • Try disabling and re-enabling the MetaMask extension
    • For mobile users, consider using MetaMask's built-in browser

USDC Contract Interaction

USDC Approval Not Working

Steps to Resolve:
  1. Balance Verification:
    • Confirm your USDC balance on Base Network
    • Ensure you're not checking balances on other networks
  2. Approval Process:
    • Try approving a smaller amount initially
    • Wait 10 minutes before retrying if a previous approval is pending
    • Verify you have sufficient ETH for the approval transaction

To transfer your funds safely:

  1. Ensure you're on Base Network in MetaMask
  2. Copy your MetaMask wallet address

Security Best Practices

Protect your investment by following these security guidelines:

  • Always verify the contract address on BaseScan before interacting

  • Keep your private keys and seed phrases secure and never share them
  • Monitor your staking activity regularly
  • Maintain sufficient ETH balance for withdrawal transactions

Transaction Safety

Follow these guidelines for safe transactions:

  • Review all transaction details carefully before confirming
  • Verify gas fees are within normal range (under $0.10)
  • Start with small test transactions when unsure
  • Keep records of your transactions for reference
  • Check smart contract verification status on BaseScan

Troubleshooting Guide

Before contacting support, please review these common issues and their solutions. Most problems can be resolved using this guide.

Transaction Issues

Transaction Pending for Long Time

When a transaction remains pending for an extended period:

  1. Check MetaMask for pending transaction notifications
  2. If the transaction is stuck:
    • Use the "Speed Up" option with a slightly higher gas price
    • Cancel the transaction if it hasn't been mined
    • Reset MetaMask activity if necessary

Note: Base Network transactions typically confirm within 15 seconds. If a transaction is pending longer than 2 minutes, action may be needed.
